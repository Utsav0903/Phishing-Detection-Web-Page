Step 1 - Open Terminal / Powershell and set python environment = python -m venv venv

Step 2 - Just after that run = .\venv\Scripts\Activate.ps1

Step 3 - install = pip install -r requirements.txt

Step 4 - Train the Model = python train_phish_model.py

Step 5 - Run the Web App = python app.py

Step 6 - Then open in browser and paste this url : = http://localhost:5000/
